## API Reference

This component is built using [Motion](https://motion.dev/) primitives and includes the following custom props:

### ShimmeringText

| Prop                                                                                                    | Type      | Default |
| ------------------------------------------------------------------------------------------------------- | --------- | ------- |
| `text` <PopoverInfo>Text to display with shimmer effect</PopoverInfo>                                   | ~string~  | -       |
| `duration` <PopoverInfo>Animation duration in seconds</PopoverInfo>                                     | ~number~  | ~2~     |
| `delay` <PopoverInfo>Delay before starting animation</PopoverInfo>                                      | ~number~  | ~0~     |
| `repeat` <PopoverInfo>Whether to repeat the animation</PopoverInfo>                                     | ~boolean~ | ~true~  |
| `repeatDelay` <PopoverInfo>Pause duration between repeats in seconds</PopoverInfo>                      | ~number~  | ~0.5~   |
| `className` <PopoverInfo>Custom CSS classes to apply to the component</PopoverInfo>                     | ~string~  | -       |
| `startOnView` <PopoverInfo>Whether to start animation when component enters viewport</PopoverInfo>      | ~boolean~ | ~true~  |
| `once` <PopoverInfo>Whether to animate only once</PopoverInfo>                                          | ~boolean~ | ~false~ |
| `inViewMargin` <PopoverInfo>Margin for in-view detection (rootMargin)</PopoverInfo>                     | ~string~  | -       |
| `spread` <PopoverInfo>Shimmer spread multiplier - affects the width of the shimmer effect</PopoverInfo> | ~number~  | ~2~     |
| `color` <PopoverInfo>Base text color (overrides default theme colors)</PopoverInfo>                     | ~string~  | -       |
| `shimmerColor` <PopoverInfo>Shimmer gradient color (overrides default theme colors)</PopoverInfo>       | ~string~  | -       |
