## API Reference

This component is built using [Motion](https://motion.dev/) primitives and includes the following custom props:

### GradientBackground

| **Prop**                                                                                            | **Type**     | **Default**                                               |
| --------------------------------------------------------------------------------------------------- | ------------ | --------------------------------------------------------- |
| `className` <PopoverInfo>CSS class for the background container.</PopoverInfo>                      | ~string~     | <Na/>                                                     |
| `transition` <PopoverInfo>Motion transition configuration for the gradient animation.</PopoverInfo> | ~Transition~ | ~\{ duration: 10, ease: 'easeInOut', repeat: Infinity \}~ |
