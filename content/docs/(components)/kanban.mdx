## API Reference

The Kanban component is a flexible drag-and-drop board for organizing items in columns. Below is the complete API reference:

### Kanban

The root Kanban component includes the following props:

| **Prop**                                                                                                                                                        | **Type**                              | **Default** |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------- | ----------- |
| `value` <PopoverInfo>Current kanban board state. The columns and their items.</PopoverInfo>                                                                     | ~Record\<string T[]>~                 | <Na/>       |
| `onValueChange` <PopoverInfo>Callback when board state changes. Called when columns/items are reordered or moved.</PopoverInfo>                                 | ~(value: Record\string T[]>) => void~ | <Na/>       |
| `getItemValue` <PopoverInfo>Function to get unique item id. Returns a unique identifier for each item.</PopoverInfo>                                            | ~(item: T) => string~                 | <Na/>       |
| `children` <PopoverInfo>Kanban board content. The board layout and columns.</PopoverInfo>                                                                       | ~React.ReactNode~                     | <Na/>       |
| `className` <PopoverInfo>Additional class names. Custom classes for the root element.</PopoverInfo>                                                             | ~string~                              | <Na/>       |
| `onMove` <PopoverInfo>Custom move handler (overrides default reordering). Called when an item is moved. If provided, disables default reordering.</PopoverInfo> | ~(event: KanbanMoveEvent) => void~    | <Na/>       |

---

### KanbanColumn

A draggable column in the Kanban board.

| **Prop**                                                                                                    | **Type**    | **Default** |
| ----------------------------------------------------------------------------------------------------------- | ----------- | ----------- |
| `value` <PopoverInfo>Column id. Unique identifier for the column.</PopoverInfo>                             | ~string~    | <Na/>       |
| `className` <PopoverInfo>Additional class names. Custom classes for the column.</PopoverInfo>               | ~string~    | <Na/>       |
| `children` <PopoverInfo>Column content. The column header and items.</PopoverInfo>                          | ~ReactNode~ | <Na/>       |
| `disabled` <PopoverInfo>Disable drag for this column. Prevents the column from being dragged.</PopoverInfo> | ~boolean~   | ~false~     |

---

### KanbanColumnHandle

A drag handle for a column.

| **Prop**                                                                                          | **Type**    | **Default** |
| ------------------------------------------------------------------------------------------------- | ----------- | ----------- |
| `asChild` <PopoverInfo>Render as child element. Use a custom element as the handle.</PopoverInfo> | ~boolean~   | ~false~     |
| `className` <PopoverInfo>Additional class names. Custom classes for the handle.</PopoverInfo>     | ~string~    | <Na/>       |
| `children` <PopoverInfo>Handle content. The handle UI.</PopoverInfo>                              | ~ReactNode~ | <Na/>       |
| `cursor` <PopoverInfo>Show grab cursor. Show grab/grabbing cursor on the handle.</PopoverInfo>    | ~boolean~   | ~true~      |

---

### KanbanItem

A draggable item in a column.

| **Prop**                                                                                                | **Type**    | **Default** |
| ------------------------------------------------------------------------------------------------------- | ----------- | ----------- |
| `value` <PopoverInfo>Item id. Unique identifier for the item.</PopoverInfo>                             | ~string~    | <Na/>       |
| `asChild` <PopoverInfo>Render as child element. Use a custom element as the item.</PopoverInfo>         | ~boolean~   | ~false~     |
| `className` <PopoverInfo>Additional class names. Custom classes for the item.</PopoverInfo>             | ~string~    | <Na/>       |
| `children` <PopoverInfo>Item content. The item UI.</PopoverInfo>                                        | ~ReactNode~ | <Na/>       |
| `disabled` <PopoverInfo>Disable drag for this item. Prevents the item from being dragged.</PopoverInfo> | ~boolean~   | ~false~     |

### KanbanColumnContent

A wrapper for the items in a column.

| **Prop**                                                                                               | **Type**    | **Default** |
| ------------------------------------------------------------------------------------------------------ | ----------- | ----------- |
| `value` <PopoverInfo>Column id. The column this content belongs to.</PopoverInfo>                      | ~string~    | <Na/>       |
| `className` <PopoverInfo>Additional class names. Custom classes for the content wrapper.</PopoverInfo> | ~string~    | <Na/>       |
| `children` <PopoverInfo>Items content. The items in the column.</PopoverInfo>                          | ~ReactNode~ | <Na/>       |

### KanbanOverlay

A custom overlay for the dragged item or column.

| **Prop**                                                                                                                             | **Type**                                                                                        | **Default** |
| ------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------- | ----------- |
| `className` <PopoverInfo>Additional class names. Custom classes for the overlay.</PopoverInfo>                                       | ~string~                                                                                        | <Na/>       |
| `children` <PopoverInfo>Overlay content or render function. The overlay UI or a function that returns overlay content.</PopoverInfo> | ~ReactNode \| (params: \{ value: UniqueIdentifier; variant: 'column' \| 'item' }) => ReactNode~ | <Na/>       |
