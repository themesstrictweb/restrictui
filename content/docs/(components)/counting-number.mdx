## API Reference

This component is built using [Motion](https://motion.dev/) primitives and includes the following custom props:

### CountingNumber

| **Prop**                                                                                            | **Type**                    | **Default** |
| --------------------------------------------------------------------------------------------------- | --------------------------- | ----------- |
| `from` <PopoverInfo>Starting number for the counting animation.</PopoverInfo>                       | ~number~                    | ~0~         |
| `to` <PopoverInfo>Target number to count to.</PopoverInfo>                                          | ~number~                    | ~100~       |
| `duration` <PopoverInfo>Animation duration in seconds.</PopoverInfo>                                | ~number~                    | ~2~         |
| `delay` <PopoverInfo>Delay before starting animation in milliseconds.</PopoverInfo>                 | ~number~                    | ~0~         |
| `className` <PopoverInfo>CSS class for the number element.</PopoverInfo>                            | ~string~                    | <Na/>       |
| `startOnView` <PopoverInfo>Whether to start animation when component enters viewport.</PopoverInfo> | ~boolean~                   | ~true~      |
| `once` <PopoverInfo>Whether to animate only once when using startOnView.</PopoverInfo>              | ~boolean~                   | ~false~     |
| `inViewMargin` <PopoverInfo>Margin for in-view detection trigger area.</PopoverInfo>                | ~string~                    | <Na/>       |
| `onComplete` <PopoverInfo>Callback function when counting animation completes.</PopoverInfo>        | ~() => void~                | <Na/>       |
| `format` <PopoverInfo>Custom formatting function for the displayed number.</PopoverInfo>            | ~(value: number) => string~ | <Na/>       |
